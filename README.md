# 🎮 Number Guessing Game of DOOM!

A hilarious and interactive number guessing game where you try to guess a random number between 1 and 20. But beware — your IQ might be judged by the United Nations! 🤡

## 🎯 Features
- Random number generation between 1 and 20
- Tracks your guess count
- Special messages for wrong answers (warning: may hurt your feelings)
- Detects clowns who type letters instead of numbers 🎪
- Saves your game data (name, guess count) in a dictionary
- Option to play again... if you dare
- United Nations called — they said your IQ lowered the global average 📉

## 🛠️ Tech Stack
- Python 3.x
- `random` library
- Pure sarcasm and dark humor
